# SteemStore HTML Project Plan

This folder contains a Steam-style storefront prototype built in plain HTML and CSS.

## Goals

- Build a storefront layout first, then connect game data later.
- Keep all game tiles empty placeholders for now.
- Keep root files easy to reach for GitHub upload.

## Folder Structure

- `projects/steemstore/steam.html`: main storefront page
- `projects/steemstore/steam.css`: visual style and responsive layout
- `projects/steemstore/PROJECT_PLAN.md`: implementation plan and roadmap

## Current State (Phase 1)

- Top navigation and hero section completed.
- Trending and offers sections completed.
- All game tiles are placeholder slots.
- Responsive layout enabled for desktop and mobile.

## Phase 2 (Next)

- Add `steam-data.json` with game metadata.
- Add `steam.js` to render tiles from JSON.
- Add basic sorting and search controls.

## Phase 3

- Add detail drawer/modal for each game tile.
- Add cart/wishlist state in localStorage.
- Add themed account panel and settings.

## Notes

- This is a learning and portfolio prototype.
- Keep branding generic and avoid using official Valve assets.
